👗 OOTD Director (나만의 TPO 코디 추천기)

오늘 당신의 완벽한 하루를 위한 맞춤 코디를 추천해 주는 웹 애플리케이션입니다.
사용자의 성별, 날씨, 시간, 상황(TPO), 선호 스타일을 분석하여 최적의 옷차림을 텍스트로 제안하고, 구글 이미지 검색과 연동하여 시각적인 레퍼런스를 즉시 제공합니다.

✨ 주요 기능

디테일한 조건 선택: 성별, 날씨(봄/가을, 여름, 겨울), 시간(낮/밤), 상황(결혼식, 장례식, 면접, 운동, 파티, 데이트, 휴가), 스타일(무난, 심플, 화려, 개성) 설정 가능.

맞춤형 코디 제안: 선택한 조건들의 조합을 분석하여 상황에 꼭 맞는 구체적인 아이템과 스타일링 팁 제공.

이미지 레퍼런스 연동: 추천된 코디에 맞는 최적의 검색어(키워드)를 자동 생성하여 구글 이미지 검색 결과로 연결.

모바일 최적화 UI: 스마트폰 환경에서도 터치하기 쉽고 직관적인 모던 디자인 적용 (Tailwind CSS 활용).

🛠 기술 스택

HTML5

Tailwind CSS (via CDN)

Vanilla JavaScript

🚀 사용 방법

옵션 메뉴에서 본인의 현재 상황(성별, 날씨, 상황 등)을 선택합니다.

코디 추천받기 버튼을 클릭합니다.

하단에 나타나는 추천 코디 텍스트를 확인합니다.

이 코디 구글에서 찾아보기 버튼을 클릭하여 실제 스타일링 이미지를 참고합니다.
