# 👗 OOTD Director (나만의 TPO 코디 추천기)

## 📌 프로젝트 개요 (Project Overview)
본 프로젝트는 사용자의 성별, 날씨, 시간대, 상황(TPO), 선호 스타일 등을 입력받아 가장 적합한 옷차림을 실시간으로 제안해 주는 **스마트 맞춤형 코디 추천 웹 애플리케이션**입니다.

## 🎯 1. 어떤 문제를 해결하려 했는지 (Problem Statement)

바쁜 현대인들은 매일 아침 "오늘 뭐 입지?"라는 고민에 상당한 시간을 소비합니다. 특히 결혼식 하객, 면접, 장례식 등 격식을 차려야 하는 자리나 특별한 데이트, 파티 등 중요한 일정이 있는 날에는 코디에 대한 스트레스가 더욱 가중됩니다. 기존의 날씨 앱이나 패션 앱들은 다음과 같은 한계가 있었습니다.

*   **TPO(Time, Place, Occasion) 반영의 부족:** 단순히 기온에 따른 옷차림만 추천할 뿐, 사용자가 어디에 가는지(목적)에 대한 맥락을 이해하지 못합니다. (예: 여름이라 하더라도 면접장에서는 반바지를 입을 수 없음)
*   **개인 취향의 무시:** 모든 사람에게 획일화된 코디를 제안하여, 화려한 스타일을 좋아하는 사람이나 무난한 스타일을 좋아하는 사람 모두를 만족시키지 못합니다.
*   **시각적 레퍼런스 탐색의 번거로움:** 텍스트로만 추천을 받으면 실제 이미지를 떠올리기 어려워, 사용자가 다시 포털 사이트를 열고 검색해야 하는 불편함이 있습니다.

**💡 해결 목표:**
본 프로젝트는 **상황(Occasion)을 최우선 기준으로 삼고 날씨, 시간, 취향을 복합적으로 필터링하여 매우 구체적이고 실용적인 코디 텍스트를 제공**하며, 추천받은 코디의 **실제 시각적 레퍼런스(이미지)를 즉각적으로 확인할 수 있는 직통 검색 기능**을 연결하여 코디 고민 시간을 획기적으로 단축시키는 것을 목표로 하였습니다.

## ✨ 2. 어떤 기능을 구현했는지 (Implemented Features)

모바일 환경에 최적화된 직관적인 UI(Tailwind CSS 적용)를 바탕으로 다음의 핵심 기능들을 구현했습니다.

### 🎛️ 5차원 다중 조건 입력 (Multi-Dimensional Input Filter)
사용자로부터 5가지 핵심 컨텍스트를 빠르고 간편하게 입력받을 수 있는 드롭다운(Select) 폼을 제공합니다.
*   `성별`: 여성 / 남성
*   `날씨`: 봄가을(선선함) / 여름(더움) / 겨울(추움)
*   `시간`: 낮 / 밤
*   `상황`: 결혼식, 장례식, 면접, 운동, 파티, 데이트, 휴가 (총 7가지 TPO 상황)
*   `스타일`: 무난함, 심플함, 화려함, 유니크함 (총 4가지 취향)

### 🧠 상황 우선순위 기반 추천 알고리즘 (Context-Aware Recommendation)
*   단순한 `if-else`의 나열이 아닌, **'상황(Occasion)'을 최상위 조건으로 두고 내부에서 날씨와 스타일을 세부 분기**하는 논리적 알고리즘을 적용했습니다. 
*   예를 들어, '장례식'이나 '면접'과 같은 엄격한 TPO가 선택되면 사용자의 '스타일(화려함 등)' 입력은 무시되거나 상황에 맞게 톤다운되며 예의와 격식을 강조하는 코디를 1순위로 제안합니다. 반면 '파티'나 '데이트'의 경우 사용자의 개성(스타일)을 극대화하는 수식어를 조합하여 동적으로 출력합니다.

### 🔍 원클릭 구글 이미지 검색 연동 (One-Click Visual Reference)
*   추천된 텍스트 결과 박스 하단에 **"이 코디 구글에서 찾아보기" 버튼**을 동적으로 생성합니다.
*   자바스크립트 로직 내부에서 추천 결과에 매칭되는 가장 정확한 **한글 검색 키워드(예: "남자 겨울 면접 정장", "여자 여름 바캉스룩")를 실시간으로 조합**하여, 구글 이미지 검색(Google Image Search) 결과 페이지로 새 탭에서 즉시 연결해 줍니다. 이를 통해 사용자는 텍스트 추천을 눈으로 바로 확인하고 쇼핑이나 코디에 참고할 수 있습니다.
